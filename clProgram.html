build

Compiles a program source, or builds program from binaries.

arguments:

  • options:: command line options, see the platform API documentation
  • devices:: array of CLDevice instances or handles or null. If it is a non-null value, the program executable is built for devices specified in this list for which a source or binary has been loaded, if it's null the program executable is built for all devices associated with program for which a source or binary has been loaded.

returns:

A promise that resolves to no value.
